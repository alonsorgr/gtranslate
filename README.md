# Traductor
## Utilidad para Ubuntu 18.04 escrita en Python
### Requisitos
- GTK 3.20
- python3
- translate-shell
### Instalación
Instale las dependencias:
```sh
$ sudo apt-get install libgtk-3-dev
$ sudo apt-get install python3
$ sudo apt-get install translate-shell
```
### Uso:
```sh
python3 gtranslate.py
```
